# Titanic Survival Prediction

End-to-end machine learning project for predicting passenger survival in the Kaggle Titanic competition

**Competition:** https://www.kaggle.com/competitions/titanic

**Kaggle Notebook:** https://www.kaggle.com/code/neckpuck/titanic-survival-prediction/notebook

## Project Goals

The main objective is to build and compare several classical machine learning models, analyze their perfomance

## Workflow

- Exploratory Data Analysis (EDA)
- Data preprocessing
- Trainig baseline models
- Model comparison
- Error analysis
- Hyperparameter tuning
- Final submission

## Models

- Stochastic Gradient Descent Classifier
- Support Vector Machine
- Random Forest
- KNN

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-score
- Precision-Recall Curve
- ROC Curve
- ROC AUC
- Confusion Matrix

Public Kaggle Score: **0.76076**
